About Pastrami

Pastrami is an audio software synth that was forked from ZynAddSubFx to improve the JACK support. It focused on supporting Linux and has since been developed further in various ways. See the history page for more details.

Pastrami is free software, licensed under GPL version 2.

The current lead developer and maintainer is Will Godfrey. For more information about the people who have contributed to the project in various ways, see the page on contributors.
